# Task Management Application

A task management system that allows users to create, update, manage, and track tasks efficiently, with task analytics features like priority distribution, completion rates, and upcoming deadlines.

## Features

- **Task Management**:
  - Create, Read, Update, Delete (CRUD) tasks.
  - Set task priority (Low, Medium, High).
  - Set due dates for tasks.
  - Mark tasks as completed or pending.
- **Task Analytics**:
  - **Task Distribution by Priority**: Visual representation of tasks divided by their priority levels using charts.
  - **Task Completion Rate**: A progress bar showing the percentage of completed tasks.
  - **Upcoming Deadlines**: List or calendar view of tasks that are nearing their due dates.
- **Responsive User Interface**:
  - Built using **Material-UI** for a sleek and modern design.
  - **Responsive layout** that adapts across devices.
- **Dashboard**:
  - All analytics and tasks in one view.
  - Clean and user-friendly navigation.
  
## Technology Stack

- **Frontend**: React, Material-UI
- **Backend**: Node.js, Express
- **Database**: MongoDB
